# 📊 Market Research & Customer Segmentation Tool

## 🚀 Project Overview
This project is an Advanced Excel dashboard designed to analyze customer data and perform segmentation for better business decision-making.

## 🎯 Objective
To identify high-value customers, analyze buying patterns, and provide actionable business insights.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Analysis
- Dashboard Design

## 📂 Dataset Features-<a href="https://github.com/shlaghit7/market-segmentation-excel-dashboard"
- Customer demographics
- Purchase behavior
- Revenue metrics
- Segmentation logic

## 📊 Dashboard Features
- KPI Cards (Customers, Revenue, AOV)
- Customer Segmentation (High / Medium / Low)
- Revenue Trend Analysis
- Interactive Charts
